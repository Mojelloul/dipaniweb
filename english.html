Our Technology Choices

An optimized stack for Dipani

Why React Native?

Open-source framework developed by Facebook to create cross-platform mobile applications (iOS and Android) with a single codebase.

Key advantages:

  • Fast & economical development → One code for two platforms
  • Smooth performance → Close to native application
  • Simplified maintenance → Less effort for bugs and new features
  • Rich community and plugins → Large ecosystem of tools
  • Scalability → Easy to scale with frequent updates

Why not use Flutter or native applications?

  • Flutter → Excellent alternative, but React Native has more developers and better compatibility
  • Native applications (Swift/Kotlin) → More expensive and time-consuming to develop

Why Supabase?

Open-source alternative to Firebase, offering a PostgreSQL database with automatic API and integrated authentication management.

Key advantages:

  • Scalable database → PostgreSQL, robust and performant
  • No-Code/Low-Code backend → Automatically generated APIs
  • Secure authentication → User management with OAuth, email/password
  • Integrated file storage → Ideal for managing images and videos
  • Open-source & Cheaper than Firebase → Data control and reduced costs

Why not use Firebase?

  • Firebase can become very expensive with user volume
  • Supabase allows better long-term scalability
  • No dependency on Google

Artificial Intelligence

Key features:

  • Text and photo analysis to precisely identify the problem
  • Advanced matching algorithm for the most qualified experts
  • Precise classification of problems and solutions
  • Personalized recommendations based on history
  • Continuous improvement through machine learning

Conclusion: An optimized stack for Dipani

  • React Native + Supabase = Speed, reduced cost, scalability, and better data management
  • Perfect for an application that needs to evolve with a growing user base
  • Best balance between performance, development ease, and cost

🚀 This technological choice allows us to quickly launch Dipani while ensuring its growth and success in the long term.
